# Bookshelf Manager - Digital library and reading list

**Package**: `@apps/bookshelf`
**Version**: 0.1.0
**Repository**: `pkg-apps-bookshelf`

## Overview

Bookshelf Manager - Digital library and reading list

## Installation

### Prerequisites

This package requires the following external commands:

- `curl`
- `jq`

### Using pkg-cli

```bash
# Install from GitHub
pkg-cli install @apps/bookshelf

# Or install from local source
cd ~/.pkgs
stow bookshelf
```

## Usage

### Application

Bookshelf Manager - Digital library and reading list

Run the application to see usage instructions.

## Configuration

Configuration files are typically stored in:
- `~/.config/bookshelf/` - User configuration
- `~/.local/share/bookshelf/` - Application data

## Uninstallation

```bash
# Using pkg-cli
pkg-cli uninstall @apps/bookshelf

# Or manual unstow
cd ~/.pkgs
stow -D bookshelf
```
